                                                üîë OAuth2 API Endpoints

These endpoints allow authentication using third-party providers (Google, Facebook, GitHub, etc.) in Laravel.

 ___________________________________________________________________________________________________________________________________________
| Method - Endpoint                              |                               Description  
|------------------------------------------------|------------------------------------------------------------------------------------------
| GET    /api/auth/{provider}/redirect           | Redirects the user to the OAuth provider (Google, Facebook, etc.).  
| GET    /api/auth/{provider}/callback           | Handles OAuth provider response, authenticates the user, and issues a Sanctum token.  
| GET    /api/auth/{provider}/user               | Retrieves the authenticated user's details.  
| POST   /api/auth/logout                        | Logs out the user and revokes the access token.  
|------------------------------------------------|------------------------------------------------------------------------------------------
| POST   /api/auth/{provider}/token-login        | Logs in the user using an OAuth access token from the provider (for mobile apps). 
| POST   /api/auth/token/refresh                 | Refreshes the OAuth2 access token (if supported by the provider). 
|________________________________________________|__________________________________________________________________________________________
